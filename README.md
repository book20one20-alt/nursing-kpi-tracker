# 🏥 نظام تتبع مؤشرات الأداء التمريضي
# Nursing KPI Tracker

> مشروع مخصص لمدراء العمليات التمريضية لمتابعة وتحليل مؤشرات الأداء الرئيسية

---

## 📋 وصف المشروع

هذا النظام يساعد مدير عمليات التمريض على:
- 📊 تتبع مؤشرات الأداء الأسبوعية والشهرية
- 👩‍⚕️ إدارة بيانات الكوادر التمريضية
- 📈 رصد معدلات الحوادث وسلامة المرضى
- 🎯 متابعة أهداف الجودة

---

## 🗂️ هيكل المشروع

```
nursing-kpi-tracker/
├── README.md              # وصف المشروع
├── data/
│   └── sample_kpis.csv    # بيانات تجريبية لمؤشرات الأداء
├── templates/
│   └── weekly_report.md   # قالب التقرير الأسبوعي
└── docs/
    └── kpi_definitions.md # تعريفات المؤشرات
```

---

## 📊 المؤشرات المتتبعة

| المؤشر | الهدف | التكرار |
|--------|-------|----------|
| معدل رضا المريض | ≥ 90% | شهري |
| نسبة الالتزام بالبروتوكولات | ≥ 95% | أسبوعي |
| معدل الحوادث لكل 1000 يوم مريض | < 2 | شهري |
| نسبة حضور الكوادر | ≥ 95% | أسبوعي |
| معدل دوران الموظفين | < 10% | ربع سنوي |
| ساعات التدريب للممرض | ≥ 20 ساعة/سنة | سنوي |

---

## 🚀 كيفية الاستخدام

1. استنسخ المستودع:
```bash
git clone https://github.com/book20one20-alt/nursing-kpi-tracker.git
```

2. افتح ملف `data/sample_kpis.csv` وأدخل بياناتك

3. استخدم قالب `templates/weekly_report.md` لإنشاء تقاريرك
